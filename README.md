# My zsh themes

## Moonsphere

Configure your theme through [Antigen]:

```shell
antigen theme timkurvers/zsh-themes moonsphere
```

### Features

- Clean / dirty working directory indicators
- Full paths (home directory collapses)
- Python virtual environments
